# Incognito
Access the world wide web

[Old repo](https://github.com/caracal-js/incognito)

# Setup
```
npm i -g pnpm tsx
git clone https://github.com/titaniumnetwork-dev/Incognito.git
cd Incognito
pnpm i
tsx server/index.ts
```
